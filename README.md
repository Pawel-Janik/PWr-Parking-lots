# PWr-Parking-lots
Progressive Web App featuring real time data about parking lots on Wroc≈Çaw Univeristy of Technology. Build Using Angular 2with TypeScript and ~~Python Flask Microframework~~ ExpressJS (mostly Angular though) 

# Live demo
http://ec2-52-7-193-110.compute-1.amazonaws.com:4200/

# Install
```
git clone https://github.com/pawelnr1/PWr-Parking-lots.git
cd PWr-Parking-lots

# Install dependencies
npm install

# start server
npm run start

# Client url: http://localhost:4200
# Application ( express ) API: http://localhost:4300
```
